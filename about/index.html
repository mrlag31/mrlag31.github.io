About page

Welcome to Yohann "MrLag" Bosqued's blog where I will discuss everything ranging from math and physics to computer science. I am currently a soon-to-be computer science engineer, specialized in computer theory. This blog has been written by hand in typescript using the following packages:

  • TypeScript used everywhere involving a web script.
  • glob to handle set of files using glob syntax.
  • jsdom for document handling outside of a browser.
  • maked to transform .md (Markdown) posts as html.
  • highlight.js to handle posts' codes.
  • marked-katex-extension (slightly modified) to create TeXTeXTeX sections in posts using KaTeX.
